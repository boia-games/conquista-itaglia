Aggiungi un giorno

Strumento locale. Non scrive nulla online: prepara l'immagine ottimizzata e il data.json aggiornato, che poi committi tu sul repo. Solo chi ha accesso in scrittura al repo può pubblicare.

1. Carica i dati attuali

Provo a leggere ./data.json dalla stessa cartella. Se apri il file con doppio-clic, caricalo a mano.

2. Esito dello scontro in sospeso

Carica prima i dati…

3. Nuovo scontro

4. Genera i file da committare

Poi, nella cartella del sito:

git add images/ data.json
git commit -m "Giorno N"
git push

Il sito pubblico su GitHub Pages si aggiorna da solo dopo il push.
